# Animous AI

Animous AI adalah web chatbot berbasis AI dengan tampilan modern seperti Gemini, menggunakan Flask, JavaScript, dan Groq API.

## Fitur

- Chat AI realtime
- Memory chat dengan localStorage
- Sidebar chat history
- Upload PDF, TXT, dan gambar
- AI vision untuk membaca gambar
- Markdown response
- Voice input
- Responsive mobile
- Clear history
- Loading spinner

## Tech Stack

- Python Flask
- HTML
- CSS
- JavaScript
- Groq API
- PyPDF2
